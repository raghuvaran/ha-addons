## Add-ons

### Spotify YouTube Sync

Automatically syncs Spotify's "Top 50 Global" playlist to your YouTube playlist.

- Runs on a configurable schedule (default: daily at 2 AM)
- Caches video IDs to minimize YouTube API quota usage
- Uses LIS algorithm for minimal playlist modifications

## Installation

1. Click the button above, or manually add this repository URL in Home Assistant:
   ```
   https://github.com/raghuvaran/ha-addons
   ```
2. Find "Spotify YouTube Sync" in the add-on store
3. Install and configure
